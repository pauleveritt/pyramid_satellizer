==================
pyramid_satellizer
==================

Investigating JWT auth between Pyramid username/passwords and Angular
using satellizer.

Server Usage
============

- Make a virtualenv

- virtualenv/bin/python ./setup.py develop

- virtualenv/bin/pserve development.ini

Browser Usage
=============

- npm install

- npm start

- Go to http://127.0.0.1:8080/src/

- Live reload is available
